# 🕵️ Detective Game  

This is a detective game. **SQL** is used to find the thief.  

## 📂 Files Overview  
- **`answers.txt`** – Contains the final answer to the game.  
- **`output.txt`** – Stores every clue gathered during the exercise.  
- **`log.sql`** – Shows my SQL code and the steps I took.  
- **`schema.txt`** – Contains information about the database.
