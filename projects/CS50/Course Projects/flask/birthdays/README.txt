# 🎉 Birthday Tracker  

A simple web application that allows users to store and view birthdays using Flask and SQLite.  

---

## 🚀 Features  
- Add a person's name and birthday (month & day) to the database.  
- View all stored birthdays on the homepage.  

---

## 🛠️ Technologies Used  
- **Python** (Flask framework)  
- **SQLite** (CS50 SQL Library)  
- **HTML & Jinja2** (for rendering templates)  

---

## 📌 Setup Instructions  

### 1️⃣ Install Dependencies
